Scroll Reveal Pattern (Webflow IX2)

Elements use data-w-id + inline style="opacity:1;transform:translate3d(0,0,0)…" set by Webflow IX2 on scroll entry. The initial state (before animation fires) is:

opacity: 0;
transform: translate3d(0px, 30px, 0px) scale3d(1,1,1) rotateX(0) rotateY(0) rotateZ(0);

On scroll into view → opacity: 1; transform: translate3d(0,0,0) with a staggered delay per element.

Floating Decorative SVGs (.banner-3-image-ab)
Positioned absolutely with rotateZ(29.666deg) and will-change: transform — animated via Webflow IX2 parallax.
